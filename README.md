# 查询百度网盘提取密码


![baidupankey示例截图](https://raw.githubusercontent.com/my-milk/baidupankey/master/example.png)

---


## 安装

---

```
npm install baidupankey -g
```


## 选项

---

-  `-u`, `--url`      **要查询的百度网盘链接**
-  `-h`, `--help`     **显示帮助信息**
-  `-v`, -`-version`  **显示版本号**


## 用法

---

```
bdp -u https://pan.baidu.com/s/xxxxx
```
